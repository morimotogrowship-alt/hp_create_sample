# JESSE BARBER デザイン仕様（design.md）

本書は `JESSE_BARBER_サイト構築仕様書.md` の唯一のデザイン情報源（配色・タイポグラフィ・装飾・モーション）。構成・要素・ページ並びは仕様書側、見た目のトークンは本書を参照する。

参照デザイン: https://stoophawg.com/ （高級メンズバーバー）。本書の数値は同サイトを実測して抽出したもの。

> 著作権についての注意（重要）：
> 流用してよいのは「デザインシステム（配色・フォント選定・余白・装飾の作法）」のみ。ロゴ（装飾スクリプトのワードマーク）、写真、文章など個別の制作物は参照サイトの著作物のため複製しないこと。ロゴ・写真はJESSE BARBER独自に制作（AI生成含む）する。

---

## 1. デザインの方向性（トーン＆ムード）

漆黒に近いチャコールを基調に、ゴシック（ブラックレター）見出しと金色のアクセントで「クラシック × ヴィンテージ × 高級理容室」を表現する。写真は暗めの木質・琥珀色の照明・革・植物といった温かみのある暗色トーンで統一し、UIは装飾的でありながら余白を広く取り、静謐で大人の男性的な空気をつくる。

キーワード: ヴィンテージバーバー / ブラックレター / ゴールドアクセント / 暗色×暖色照明 / 余白多め / 静謐・上質。

## 2. カラーパレット

実測値（hex）。CSS変数名は実装の推奨。

- `--color-bg`：#1C1D1F（ベース背景。漆黒に近いチャコール）
- `--color-bg-elevated`：#202124（ヘッダー等のわずかに明るい面）
- `--color-overlay`：rgba(28, 29, 31, 0.95)（固定ヘッダー/オーバーレイの半透明背景）
- `--color-text`：#E8EAED（本文・見出しの基本色。ライトグレー／オフホワイト）
- `--color-text-muted`：#B8BBBF（補足テキスト。#E8EAEDをやや暗くした想定値・調整可）
- `--color-accent`：#F3A70D（ゴールド／アンバー。サブ見出し・罫線・ボタン枠・リンクホバー）
- `--color-accent-soft`：rgba(243, 167, 13, 0.6)（淡いゴールド。ホバー下線等）
- `--color-line`：rgba(232, 234, 237, 0.15)（薄い区切り罫）

配色ルール:
- 画面はほぼ全面 `--color-bg`。カードや帯で `--color-bg-elevated` を使い分ける。
- ゴールド（`--color-accent`）は「サブ見出し」「セクションタイトル下線」「ボタン枠」「リンクホバー」に限定し、多用しない（高級感のため挿し色に留める）。
- 大見出しはゴールドではなく `--color-text`（オフホワイト）。装飾の下線にゴールドを使う。
- 純白（#FFFFFF）は使わず、#E8EAED を最も明るい色とする。

---

## 3. 背景テクスチャ

ベース背景 `--color-bg` の上に、ごく低コントラストのスワール／マーブル状テクスチャを敷く（黒〜濃グレーの流れる模様）。視認性を損なわない程度（不透明度の目安 0.04〜0.08）で全面に固定的に配置する。

- 実装：背景画像（暗色マーブル）を `background` に重ねる、または微細なノイズ/グラデーションで再現。
- テクスチャ画像はAI生成等で独自に用意（参照サイトの画像は使わない）。

## 4. タイポグラフィ

### 4.1 フォントファミリー（Google Fonts）

- 見出し・ナビ・ボタン（ディスプレイ）：`"Grenze Gotisch"`（ブラックレター系ゴシック表示書体）
- 本文（日本語・英数）：`"Noto Sans JP"`
- 補助セリフ（任意・ロゴ近傍の英字装飾等）：`"Playfair Display SC"`（参照サイトのベースに存在。使う場合は限定的に）

読み込み例（実装メモ）:
- Grenze Gotisch / Noto Sans JP / Playfair Display SC を Google Fonts から読み込む。日本語本文の可読性のため Noto Sans JP は 400/500 を用意。

### 4.2 スケール・指定（実測）

- セクション大見出し（H2相当）：Grenze Gotisch / 46px / weight 400 / letter-spacing 1px / color `--color-text` / 下に 1px ゴールド罫（後述5.1）
- サブ見出し（H3相当）：Grenze Gotisch / 30px / weight 400 / letter-spacing 1px / color `--color-accent`（ゴールド）
- ナビ項目：Grenze Gotisch / 16px / weight 400 / letter-spacing 0.5px / color `--color-text`
- 本文：Noto Sans JP / 16px / line-height 2.0（32px）/ color `--color-text` / letter-spacing normal
- ボタン文言：Grenze Gotisch / 16px / letter-spacing 0.5px

指針:
- 見出しは必ず Grenze Gotisch。英字主体だが日本語見出しが必要な場合は Noto Sans JP の中字で代替し、英字併記でブラックレターの雰囲気を保つ。
- 本文は行間を広め（2.0前後）にとり、暗背景での可読性と上質感を確保する。
- レスポンシブ：H2は SP で 32〜36px、H3は 22〜24px に縮小（clamp推奨）。

## 5. 装飾・コンポーネント

### 5.1 セクションタイトル

- 中央寄せの大見出し（Grenze Gotisch / オフホワイト）。
- 見出し直下に 1px ソリッドのゴールド罫（`border-bottom: 1px solid #F3A70D`）を、見出し幅程度の短い下線として配置。

### 5.2 ボタン（CTA：BOOK / Book Now 等）

- 背景：透明（`transparent`）
- 文字色：`--color-text`（#E8EAED）
- 枠線：`1px solid #F3A70D`（ゴールド）
- 角丸：0px（直角・シャープ）
- フォント：Grenze Gotisch / 16px / letter-spacing 0.5px
- パディング：約 5px 16px（実測 4.8px 16px。タップ領域確保のためSPは縦を増やす）
- ホバー：背景にゴールドを薄く敷く（rgba(243,167,13,0.12)）または文字色をゴールド化。トランジション 0.2〜0.3s。

### 5.3 ナビゲーション

- 項目：Grenze Gotisch / 16px / オフホワイト / letter-spacing 0.5px。
- ホバー：ゴールドのアンダーライン演出（`--color-accent` の下線をスライドイン）。
- 固定ヘッダー時の背景：`--color-overlay`（rgba(28,29,31,0.95)）。

> ナビ配置について：参照サイトは「上部中央ロゴ＋水平ナビ（固定ヘッダー）」。仕様書（4章）は「PC右端の縦型サイドバー」を指定。配置は仕様書側を優先しつつ、配色・フォント・ホバー演出（ゴールド下線）は本書に準拠する。配置をどちらに合わせるかは要確認。

### 5.4 ロゴ

- 参照サイトは装飾フローリッシュ付きの手書きスクリプト・ワードマーク（白）。
- JESSE BARBER 用に独自制作する（複製不可）。トーンは「ヴィンテージ・スクリプト＋装飾フレーム、白またはオフホワイト、暗背景前提」。

### 5.5 画像・写真

- 暗色×暖色照明で統一（木質・革・真鍮・植物・施術風景）。
- 角丸なし〜ごく小（直角基調）。必要に応じて薄いゴールドの枠や影は控えめに。
- すべてAI生成等で独自に用意（参照画像は使わない）。各画像に「ファイル名／alt／説明／キャプション」を設定（仕様書6章準拠）。

---

## 6. レイアウト・余白

- 背景は全面ダーク、コンテンツは中央寄せの広い余白で構成。
- セクション間は十分な縦マージン（PCで上下 96〜140px 目安）を取り、ゆったり見せる。
- 2カラム（テキスト＋写真）構成を多用。SPは1カラムに段組み解除。
- 直角基調（border-radius 0）でクラシックに。丸みは原則使わない。

---

## 7. モーション

- 控えめで上品に。要素のフェードイン＋わずかな上方向スライド（translateY 8〜16px）をスクロールインで適用。
- ナビ/リンクのホバーはゴールド下線のスライドイン（0.2〜0.3s ease）。
- ギャラリーは横スクロール。過度なパララックスや派手な動きは避ける。
- 初期ローディング（仕様書4章）：背景 `--color-bg`（漆黒チャコール）、ロゴはオフホワイト、アクセントにゴールド。

```css
:root{
  --color-bg:#1C1D1F;
  --color-bg-elevated:#202124;
  --color-overlay:rgba(28,29,31,.95);
  --color-text:#E8EAED;
  --color-text-muted:#B8BBBF;
  --color-accent:#F3A70D;
  --color-accent-soft:rgba(243,167,13,.6);
  --color-line:rgba(232,234,237,.15);
  --font-display:"Grenze Gotisch", sans-serif;
  --font-body:"Noto Sans JP", sans-serif;
  --font-serif:"Playfair Display SC", serif;
  --radius:0px;
  --lh-body:2;
}
```
